# 寻道大千砍树游戏

一个基于HTML、CSS和JavaScript的静态网页版砍树修仙游戏。

## 游戏介绍

寻道大千砍树是一款单机版的砍树修仙游戏，玩家通过点击树木获取资源，升级境界和装备，解锁道童和灵宠，挑战BOSS，体验完整的修仙砍树游戏。

## 功能特性

### 核心玩法
- **砍树系统**：点击树木或按空格键进行砍树，获取木材、修为和灵石
- **境界系统**：通过积累修为突破境界，提升属性
- **装备系统**：强化斧头，提升攻击力
- **道童系统**：招募道童自动砍树，提高资源获取效率
- **灵宠系统**：解锁灵宠，获得特殊技能和属性加成
- **挑战系统**：挑战不同境界的BOSS，获得稀有奖励
- **商店系统**：购买道具，提升游戏体验
- **洞府系统**：建造建筑，获得离线收益
- **多周目系统**：转世重修，保留部分属性重新开始

### 技术特性
- **纯静态HTML**：单页面应用，无需后端
- **LocalStorage存档**：自动保存游戏进度
- **响应式设计**：适配不同设备屏幕
- **表格化UI**：简洁清晰的界面布局
- **键盘操作**：支持空格键/回车键快速砍树
- **图标化界面**：使用emoji图标增强视觉效果，提升用户体验

### 已知问题修复
- **空格键滚动问题**：修复了按空格键时页面向下滚动的问题
- **修为负数问题**：修复了突破境界时修为可能变成负数的问题
- **突破条件检查**：修复了修为不足时也能突破境界的问题

## 如何运行

1. **克隆或下载**本项目到本地
2. **启动本地服务器**：
   - 方法一：使用Python内置服务器
     ```bash
     python3 -m http.server 8000
     ```
   - 方法二：使用其他本地服务器工具（如Live Server、http-server等）
3. **打开浏览器**访问 `http://localhost:8000` 开始游戏

## 游戏操作

- **鼠标点击**：点击树木区域进行砍树
- **空格键/回车键**：快速执行砍树操作
- **底部功能栏**：点击按钮打开各系统界面
- **系统表格**：通过表格内的按钮进行各种操作

## 游戏系统

### 境界系统
- 凡人一阶 → 凡人九阶 → 炼气期 → 筑基期 → 金丹期 → 元婴期 → 化神期
- 每突破一次境界，攻击力+1

### 装备系统
- 斧头：提升攻击力
- 道袍：提升生命值和木材获取
- 戒指：提升攻速和灵石获取
- 护符：提升幸运值和修为获取

### 道童系统
- 砍柴童子：自动砍树1次/秒
- 伐木匠人：自动砍树3次/秒，木材+10%
- 灵木师：自动砍树5次/秒，双倍木材

### 挑战系统
- 树妖（炼气期解锁）：掉落绿装和灵石
- 木灵精（筑基期解锁）：掉落蓝装和灵宠蛋
- 森林霸主（金丹期解锁）：掉落紫装和突破丹

### 商店系统
- 突破丹：境界突破必备
- 强化石：装备强化材料
- 延寿丹：增加寿元
- 灵宠经验丹：灵宠升级材料

## 存档系统

- **自动保存**：每30秒自动保存游戏进度
- **导出存档**：复制存档代码，可在其他设备恢复游戏
- **导入存档**：粘贴存档代码，恢复游戏进度
- **转世重修**：保留50%属性，重新开始游戏
- **重置游戏**：清空所有游戏数据

## 开发技术

- **HTML5**：页面结构
- **CSS3**：样式设计
- **JavaScript**：游戏逻辑
- **LocalStorage**：本地存储
